# Corellon's Crown

Corellon's Crown is [[People/NPCs/Fala Lefaliir|Fala Lefaliir's]] herbalist shop in [[Places/Trollskull Alley]].

## Description

- A stately three-story townhouse.
- The third floor is a bright greenhouse visible from the street.
- The shop contains herbs, aromatics, flowers, and potions.

## Session 4

- Aladeen visited looking for something to improve Trollskull Manor's smell.
- Fala gifted sage and a lavender bundle to help with the manor.
