## Transcript Pass Checklist

- Create or finalize the session page.
- Record start state and end state.
- Split actual events from planned events.
- Update involved PCs, NPCs, places, factions, objects, open threads, and dashboards.
- Mark reveal status for handouts and private information.
- Add unresolved questions and next-session hooks.
- Update [[INDEX]], [[CHANGELOG]], and this map if the source pairing changes.
