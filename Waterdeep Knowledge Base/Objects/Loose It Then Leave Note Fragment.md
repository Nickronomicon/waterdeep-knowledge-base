# Loose It Then Leave Note Fragment

The Loose It Then Leave note fragment is a mostly burned clue found near the cracked-open grell crate in the [[Places/Fish Street Warehouse]].

## Revealed Contents

The readable line says: "Loose it, then leave."

## Current Significance

This confirms that the grell was intentionally released and that the handlers were expected to abandon the site afterward. It supports [[People/NPCs/Vajra Safahr|Vajra's]] claim that Xanathar-linked actors use monster releases to spread panic and pull the Watch thin.
