# Remix Eye Heists

The Remix turns villain lairs into heist targets. The party should usually know the score, gather information, perform surveillance, prepare, then execute with complications rather than binary failure.

## Heist Procedure

1. Identify the score: which Eye, Stone, report, or record is being targeted.
2. Gather information: blueprints, staff routines, security, events, allies, escape routes.
3. Conduct onsite surveillance.
4. Do prep work: bribes, disguises, supplies, forged papers, alternate entrances, distraction plans.
5. Run the operation. Failed checks should create complications, not instantly erase the plan.
