## Trollskull Source Note

The original Trollskull chapter is now available at `../raw/waterdeep_dragon_heist/02.Trollskull-Alley.md`. Use it for neighborhood reference, Lif's original baseline, guild/faction mission scaffolding, and alley businesses, but preserve existing table truth for this campaign's customized Trollskull history, Lif seance answers, Underbough hiring, Frewn's escalation, and the smuggler's passage.

## Source Priority Reminder

Original adventure material should not override:

1. direct user instruction;
2. established table truth;
3. DM prep/custom campaign direction;
4. `../raw/Dragon-Heist-Remix.pdf`.

Use it when the Remix calls out a chapter, when a map/stat block/location baseline is needed, or when active prep needs Waterdeep setting support.
